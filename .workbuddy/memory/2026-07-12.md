# 2026-07-12 工作日志

## client 子项目卡死诊断

用户报告 client 备份进程执行后卡死，System I/O 飙高，日志停在 `[VolumeMapper] Volume C:`。

### 诊断结论：三层问题叠加

1. **日志静默（P0）**：`VolumeMapper.cpp:319` 的日志消息含 Unicode 箭头 `→` (U+2192)，是全日志第一个非 ASCII 字符。`Logger.cpp:49` 的 `std::wofstream` 未设置 UTF-8 locale，写 `→` 时 codecvt 转换失败 → failbit → 后续所有日志写入静默丢弃。进程实际仍在运行但无输出。

2. **VSS 无超时等待（P0）**：`VssManager.cpp:896` 的 `pAsync->Wait()` 无超时参数。磁盘有两个 BitLocker 加密分区，VSS 协调 BitLocker Writer 时可能无限期阻塞，System 进程 I/O 飙高。

3. **300GB 全盘双重裸读（P1）**：`BackupEngine.cpp:437,931` 的 `CreateFileW` flags=0，无 `SEQUENTIAL_SCAN`/`NO_BUFFERING`。BuildManifest 读一遍 300GB，TransferBlocks 再读一遍。每块分配 1MB 缓冲，307,200 次 DEBUG 日志。
